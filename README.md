# Word Race

## Project Description
A social space for two people to write together, view and talk about each others work, and motivate each other.

## Feature list
* Two synchronized text boxes, one accessible to each user
* User set timer
* User input goal
* 2 modes
  * Competitive
  * Commiserate
* Store user inputs
* Stretch
  * chat
  * scoreboard per session
  * user stats
